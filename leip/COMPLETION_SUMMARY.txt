Step 1: Setup
    cd c:\Users\wale8\Desktop\Investigation\ software\leip
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt
    cp .env.example .env

Step 2: Verify Installation
    python init.py
    
Expected output:
    ✓ Face Recognition module - PASS
    ✓ YOLO Detection module - PASS
    ✓ CCTV Processor module - PASS
    ✓ License Plate Recognizer - PASS
    ✓ FastAPI Backend - PASS

Step 3: Start API (Terminal 1)
    python -m uvicorn app.api:app --reload

    http://localhost:8000           ← API
    http://localhost:8000/docs      ← API Documentation

    streamlit run frontend/app.py

    http://localhost:8501           ← Dashboard

    3. Process a video with "CCTV Analysis"

    5. View results in real-time
